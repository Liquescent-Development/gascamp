# Gas Camp

**Status: design phase.** Nothing here runs yet.

Gas Camp is a lightweight sibling to [Gas City](https://github.com/gastownhall/gascity):
multi-agent orchestration sized for small, everyday tasks. One tiny Rust daemon,
a file-based ledger, Claude Code as the agent runtime, and zero CPU when idle.
Camp for lunch, city for fleets — and a documented migration path from one to
the other.

Start with the design document: [`docs/design/2026-07-05-gas-camp-design.md`](docs/design/2026-07-05-gas-camp-design.md).
